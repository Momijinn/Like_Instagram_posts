# Like_Instagram_posts

Instagram のタイムラインに表示されている投稿にいいねをする Chrome Extension.

## Usage

### requirement

- node v16.15.0

### build

```
$ npm install
```

#### debug

```
$ npm run watch
```

### build

```
$ npm run build
```

dist に生成物ができる
